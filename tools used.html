Tools used

Trello Board - Trello

Link to Soupafood’s Trello Board: Building IT Systems Board

For our project, we have decided to use Trello to act as our bulletin board. Our two main uses for Trello have been to assign tasks to each member weekly and to provide the team updates on their current progress.

Microsoft Teams - Microsoft Teams

Microsoft Teams has been our means of communication for the team. We have used it to message each other for help and host virtual meetings which have been vital to our project. We have hosted on average 2 meetings per week in order to discuss and give updates on their progress as well as to assign new tasks.

Canvas - RMIT Canvas

RMIT Canvas has been our way of getting updates on our assignment details from the tutors.

Figma - Figma

Figma is a program which allows us to design and prototype applications for mobile. This has been the key to designing and developing the visual features and functionality of our app Soupafoods.

LucidChart - LucidChart

Our SCD: Link to our SCD

LucidCharts is a program which we used to create our System Context Diagram. It allows us to create a chart using bubbles and arrows.

AWS - Amazon Web Services

AWS is a page hosting website which allows users to upload html files and then host it to create a webpage.

Google Docs - Google Docs

Google Docs is a word processor which has allowed our group to simultaneously work on a document in real time. Combined with the use of Google Drive, we have been able to share our word documents with one another by placing the files in a shared folder.


GitHub - GitHub

Link to Soupafoods A2 repository: Soupafoods A2 repository

GitHub is the world's largest code storage website and open source community, where you can save completed code or code that needs to be tested.
