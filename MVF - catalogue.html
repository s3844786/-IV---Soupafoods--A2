Scenarios & Explaination

Scenario

Fred is a university student currently studying at home, due to the COVID-19 situation, and would like to reduce the time he is in the supermarket. He scrolls through Google Play Store on his android phone and spots an interesting application named called “Soupafoods”. He lets out a sigh of relief and starts to laugh at the name of the application and spots that it is a grocery application which he happens to be interested in, which resulted in him downloading it.

As he opens the application, he goes through the tutorial to teach him the basics of the app. As he finishes setting up the application, he navigates to the catalogue section as his main goal is to use the application to help reduce the time in the shops by seeing what is on promotion at each grocery store. As he enters the catalog section, he is able to access his local Woolworths and Coles catalogs for the current week and the following week. Out of his curiosity, he is able to add other grocery stores such as ALDI to his “customised” catalogue section with the addition of comparing the prices between two supermarkets of choice.

Reasoning

The goals of these users all use the minimal feature that the application has to offer which is a designated catalogue section which can be used to access the grocery store catalogues in the current week or the following week. Within this section, the user is able to customise to their personal preference such as adding/removing grocery catalogues they would like to see, compare items at two different supermarkets to understand which one sells it for cheaper and the ability to add the item to the shopping list.

This will help save the users time to be used elsewhere in addition to providing a reminder of the new catalogue being released and the primary purpose to help the user to save money as the being a customer, we would love to save our money and spend it on other items.


User Stories

As a parent, I want to check out the catalogues so that I have an idea of what I will most likely purchase for the family. As a university student that has a low income, I want to be able to access the catalogues from different supermarkets so that I can save money and time. As a customer, I want to view promotional catalogs and compare them with other supermarkets so that I am able to maximise my time and compare prices so I can decide where to go to do my shopping.


Validation criteria

Scenario: Where the use miss clicks on the store catalogue they would like to visit

Given that the parent is within the catalogue section and want to view the promotional catalogue, the user misclicks to view the wrong catalogue of their choice and then would like to hop out of the catalogue they are viewing through a “back” button or the back button on their phone to access the catalogue of their choice.

Scenario: 

Given that the university student is in the catalogue section and wants to view other products that are on discount, when the user clicks on the week/fortnight catalogue specials, Explore more products, then they are able to search for more items that are on special based on the current week.

Scenario: 

Given that the customer is in the catalogue sections and wants to compare items from different grocery stores when they have the ability to then they will be able to maximise the use of the applicationin order to fulfill what they want.


Design Diagrams

Whole design of the promotional catalogue section

Main section (Figure 1)

This is the main section of the “promotional catalogue” section of the application. In this section of the app, the user is able to go back if they would like to go to a different section (whether it's recipes, promotional catalogue, shopping list, expiry date tracker, your pantry or connectivity).

In Figure 1, the user is able to go back through the logo button, or click on the grocery store of their choice in order to view the current week's catalogue and the next week's one. The add button will allow the user to add the grocery store of their choice and the compare items button will allow the user to compare items between the stores of their choice.

Add stores section (Figure 2)

If the user decides to click on the floating action button (FAB) which is the add button, it will bring them to Figure 2.

In Figure 2, the user is able to click the back button if they would like to go back to where they were brought from. Otherwise, they are able to search for the stores they would like to add through the search bar and add to their personal section in the main catalogue section.

Viewing catalogues section (Figure 3)

In this section of the application, the user is able to view the catalogue through Figure 1, when they click on the grocery store of their choice.

In Figure 3, the user is able to view this weeks and the following week catalogue and be viewed in the application itself. If the user would like to hop out of this section, they are able to click on the back button which will bring them back to the main catalogue section, Figure 1.

Choosing item to compare section (Figure 4)

Following the design from Figure 1 (main section of the promotional catalogue), if the user decides to click on compare items between stores, it will bring them to Figure 4 where the user is able to search for the item they would like to compare.

In addition, this section of the application has sections where the user searches such as Fruit and Vegetables, in order to find the item they would like to compare as well as suggested items that they have searched before or are on their shopping list.

Item search section (Figure 5)

If the user decides to use the search bar to search for the item they would like to compare, the user interface would look like Figure 5. In Figure 5, the search will be filtered to what the user has searched in the search bar where the user can select the item they would like to compare using the compare button.

Price comparison section (Figure 6)

In Figure 6, it will show the item that the user has decided to compare in the search bar along with the different types of variations of the type of item such as white rice, homebrand or sun rice.
