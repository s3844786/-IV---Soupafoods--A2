Introduction

The app will have a function labeled “Recipes.” The function of this feature is to present the customer with recipes for the items they currently have in their pantry. Here is how the sequence for viewing recipes would work in the app.

User stories + Validation criteria

User story + Validation criteria 1

As a parent, I want to be able too find recipes based on my ingredients, so I can learn to make a different type of dish based on the ingredients i have

Given that I'm a parent and have installed the Soupafoods application on my phone,

When I go on the recipe section of the application and search for reciepes that fit my criteria,

Then the application will show to the parent what recipes they are able to make.

User story + Validation criteria 2

As a university student, I want to be able to identify recipes that can be easily made with the ingredients I have and limited money, so I can have different type of meals and food on different days.

Given that I am a university student that has a job while studying, 

I want to be able to find recipes with the ingredients that I have bought from the store,

Then I can be able to enjoy my meals and have the satisfaction of making different type of meals.

Design diagrams

Firstly, the user will click on “Recipes” on the home-page.

After that, the user is greeted with three recipes the app has randomly chosen from the vast amount of recipes that the app has acquired. The user can scroll down to see more recipes. The app has taken pre-existing recipes as well as the app’s own custom recipes that were added during development. Under the title of the recipe, there are tags that define the recipe.

The user is given the chance to search for specific recipes, by typing into the search bar,

On this page, the user is able to select what type of meal they want the recipe for (i.e., breakfast, lunch, dinner, etc. ). They can select any dietary restrictions they might have, such following a keto diet or being vegan. Furthermore, the user can choose what cuisine they want the recipes for, such as Asian, Indian, or Chinese. Lastly, the user is able to choose the difficulty of the recipe as well.

The user is given the opportunity to view recipes based on their own preferences. They can access the preferences by clicking on the preferences button on the recipes page. They will be taken to the following page.
